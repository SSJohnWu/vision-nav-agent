# Product Shopper

辨識圖片中的商品並自動搜尋蝦皮（shopee.com.tw）取得價格與購買連結。

## 觸發條件
- 使用者說「拍照」並希望購買圖中商品
- 需要找出商品在蝦皮上的價格與連結

## 操作流程

### Step 1: 分析圖片商品
使用視覺模型分析圖片，輸出：
- 商品名稱（繁體中文）
- 品牌（如果有）
- 商品特徵（顏色、尺寸等）

Prompt 格式：
```
請仔細描述這張圖片中的商品，包含：
1. 商品名稱
2. 品牌（如果有）
3. 商品特徵（顏色、尺寸、包裝等）
只回傳商品資訊，不要其他多餘描述。
```

### Step 2: 搜尋蝦皮
使用 `ollama_web_search` 搜尋：
```
site:shopee.com.tw [商品名称]
```

取前三個結果，取得：
- 商品標題
- 商品連結
- 價格區間

### Step 3: 取得商品詳情
對每個搜尋結果使用 `ollama_web_fetch` 取得商品頁面，確認：
- 實際價格
- 商品評價
- 銷量

### Step 4: 輸出結果
輸出格式：
```
🛒 已找到商品：[商品名稱]
💰 最低價格：[價格]
⭐ 評價：[評價數]
🔗 購買連結：[連結]
```

## 重要提醒
- 蝦皮需要登入才能加入購物車，系統會提供連結由使用者自行點擊加入
- 優先選擇評價高、销量好的商品
- 價格已含運費的優先

## 工具依賴
- `ollama_web_search`: 搜尋網頁
- `ollama_web_fetch`: 取得網頁內容
- Vision model (gemini/llava) 分析圖片
